# Deploying with Deeploy

Deeploy (https://deeploy.ratio1.ai) is Ratio1's managed deployment layer for
Cloud Service Providers (CSPs). It combines familiar container workflow concepts
(projects, jobs, templates, and runtime controls) with blockchain-backed
identity, escrowed payments, and oracle-coordinated scheduling.

Use this section to move from onboarding and first deployment to day-2
operations for production workloads.
