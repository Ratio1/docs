# First deploy

This page covers the first deployment flow after account setup and Deeploy
Secrets configuration.

## Step 1: Create your first Project and Job

1. Create a project for the app/environment you want to run.
2. Add a job from a suitable template (or a generic app profile).
3. Select runtime parameters (CPU, memory, optional GPU, region/node scope).
4. Configure network exposure (port mapping and tunnel/domain settings).

## Step 2: Fund and deploy

1. Review estimated cost and duration.
2. Approve USDC spending if prompted.
3. Confirm `Pay & Deploy` to lock escrow and start orchestration.

## Step 3: Verify post-deploy state

After deployment starts, validate:

- Job status transitions to running.
- Health checks and logs are available.
- External endpoint resolves through your configured tunnel/domain.
