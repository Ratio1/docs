# Nodes and Licenses

In the Explorer, **Nodes** and **Licenses** are separate pages:

- Nodes: **https://explorer.ratio1.ai/**
- Licenses: **https://explorer.ratio1.ai/licenses**

This guide covers both pages and how to drill into detail views from each.

Use it to answer questions like:

- Which nodes are currently online and healthy?
- Which license powers a specific node?
- What is the economic state of a license (ownership, allocated supply,
  claimable rewards)?

---

## Nodes page

The Nodes page lists all nodes and their core operational data, including:

- node alias
- node addresses
- linked operator / owner identity
- linked license
- current status (online/offline/active) and availability
- region or location label
- version

Use this page for fast filtering and to find a specific node.

---

### Node detail view

Selecting a node from `https://explorer.ratio1.ai/` opens that node's detail
page, where you can inspect:

- hardware profile (CPU, GPU, RAM, storage)
- location / geography
- oracle flag (yes/no)
- data center flag (yes/no)
- KYB flag (yes/no)
- availability indicators on Ratio1
- recent activity and health timeline

Use this page when validating whether a node is suitable for production
workloads or when troubleshooting reliability issues.

## Licenses page

The Licenses page lists all licenses and their core data, including:

- license identifier
- license type (GND / MND / ND)
- owner / wallet
- linked node (if linked)
- linking timestamp

Use this page for a fast overview of licensed capacity distribution.

---

### License detail view

Selecting a license from `https://explorer.ratio1.ai/licenses` opens the license
detail page, where you can inspect license economics such as:

- allocated R1 supply
- accumulated and claimable rewards
- current linkage and activity state

Use this view to evaluate the financial state of a license and track reward
flows over time.
