# Stats and Analytics

The **Stats and Analytics** page is:

- **https://explorer.ratio1.ai/stats**

Use it for executive monitoring, adoption tracking, and trend validation across
the network.

---

## Live dashboard structure

The `/stats` page is a live dashboard. Values change continuously as network
activity changes.

The sections below match what is displayed on the page.

---

### Token

The Token section provides a live overview of supply dynamics, including:

- Circulating Supply
- Total Supply
- Max. Theoretical Supply
- Burned

---

### PoAI

The PoAI section tracks CSP-funded workload economics and activity:

- Escrow TVL (`$USDC`)
- Total Active Jobs
- Daily PoAI Rewards (`$USDC`)
- Daily PoAI Burn (`$R1`)

---

### Adoption

The Adoption section includes:

- Total Overall Adoption
- ND Sales
- PoAI Volume

This chart is important for understanding the **claiming limits for MND
allocation**, because the claimable portion is tied to network adoption
progress.

---

### Treasury Wallets

Treasury wallet breakdown fields:

- Wallet
- % of GND
- Mined / To be mined
- `$R1` Balance
- `$R1` Transferred Out
- `$USDC` Balance

You can use this section to compare treasury allocation, mining progress, and
transfer behavior across treasury buckets (for example Grants, LP, CSR,
Expenses, Marketing, and VAT).

---

### Nodes (geography and compliance)

Nodes geography table fields:

- Location
- Total Count
- Data Center Nodes
- KYC / KYB

Use this section to understand node distribution by country, data-center share,
and KYC/KYB composition.
